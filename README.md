# pixelblaze
Pixelblaze animation scripts for LEDs connected with the ElectroMage PixelBlaze or PixelBlaze Pico [processors](https://electromage.com/pixelblaze).  Note these files have a .js extension, however the scripting language is not a full implementation of javascript.  You can find the language reference [here](https://electromage.com/docs/language-reference).

Prefixes: 
* Motion: uses the accelerometer to alter animations
* Sound: reacts to audio input (must have the audio expansion board for this to work)
